# AWTRIX2.0_APPS_jar-format
Collect AWTRIX2.0 native apps in jar format

There are download from appstore of awtrix2.0 

or compile by B4J IDE with https://github.com/awtrix/AWTRIX2.0-Apps

Hope it works for you
